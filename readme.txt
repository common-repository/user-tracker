== Description ==
Track the pages and posts visited by users, without using external servers. Displays a table with users and page views count. You can view detail for visited pages and posts


== Installation ==
1. Upload `user-tracker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View tracked pages in 'users' menu
